# Zoho CRM Webform Designer - User Manual

## Introduction
The **Zoho CRM Webform Designer** is a powerful tool designed to customize and style your Zoho CRM webforms. It allows you to transform basic HTML forms into professional, branded, and visually appealing interfaces without writing code.

---

## Editor Interface Overview
The editor is divided into logical sections, each controlling a specific aspect of your form's design. Use the sidebar navigation to switch between these settings categories:

1.  **Form Settings**: Global styles for the form container.
2.  **Header Settings**: Title and subtitle configuration.
3.  **Logo Settings**: Brand logo placement and sizing.
4.  **Layout Settings**: Spacing and alignment of fields.
5.  **Label Settings**: Field label styling.
6.  **Input Settings**: Input field styling.
7.  **Button Settings**: Submit and Reset button customization.

---

## Detailed Configuration Options

### 1. Form Settings
Control the overall appearance of the form container.

*   **Background Type**: Choose between **Solid** color or **Gradient**.
    *   *Solid*: Select a single background color.
    *   *Gradient*: Select Start and End colors, and adjust the **Gradient Angle** (0-360Â°).
*   **Padding**: Adjust the space between the form content and the outer edges (10px - 60px).
*   **Row Spacing**: Control vertical space between form rows (0px - 40px).
*   **Border Radius**: Round the corners of the form container (0px - 30px).
*   **Border Settings**:
    *   **Border Width**: Set thickness (0px - 10px).
    *   **Border Color**: Choose the color of the border.
*   **Shadow**: Apply a drop shadow depth (None, Small, Medium, Large, XL, 2XL).
*   **Font Family**: Select the global font for the form (e.g., Helvetica, Inter, Roboto, etc.).

### 2. Header Settings
Customize the top section of your form.

*   **Show Header**: Toggle to enable/disable. *Note: If your original form HTML contained a header, this may be locked to 'Visible'.*
*   **Header Text**: Enter your main title (e.g., "Contact Us").
*   **Background**: Independent background control (Solid or Gradient).
*   **Typography**:
    *   **Text Color**: Color of the main title.
    *   **Font Size**: 16px - 40px.
    *   **Font Weight**: Light (300) to Extra Bold (800).
    *   **Font Family**: Distinct font selection for the header.
*   **Subheader**: Optional subtitle text.
    *   **Show Subheader**: Toggle visibility.
    *   **Text**: Enter subtitle content.
    *   **Color, Size (10-24px), Weight**: Customize independently.
    *   **Gap**: Spacing between the main header and subheader.

### 3. Logo Settings
Add your company branding.

*   **Show Logo**: Toggle visibility.
*   **Logo URL**: Direct link to your logo image file (e.g., `https://example.com/logo.png`).
*   **Logo Size**: Select **Small**, **Medium**, or **Large**.
*   **Position**: Place the logo in one of 6 positions:
    *   Top Left (TL), Top Center (TC), Top Right (TR)
    *   Bottom Left (BL), Bottom Center (BC), Bottom Right (BR)

### 4. Layout Settings
Fine-tune the alignment and spacing of your form fields.

*   **Label Width**: Percentage width for field labels (20% - 50%).
*   **Input Width**: Percentage width for input fields (50% - 80%).
*   **Row Spacing**: Vertical distance between separate fields.
*   **Field Spacing**: Horizontal distance between the label and the input box.
*   **Reset to Default**: Quickly restore standard layout spacing.

### 5. Label Settings
Style the text labels next to your input fields.

*   **Text Color**: Color of the label text.
*   **Font Size**: 10px - 20px.
*   **Font Weight**: Light to Extra Bold.
*   **Label Width**: Shortcut to the same setting in Layout Settings.
*   **Required Indicator Color**: Change the color of the asterisk (*) denoting mandatory fields.

### 6. Input Settings
Style the text boxes where users enter data.

*   **Background Color**: Background of the input field itself.
*   **Border Settings**:
    *   **Color**: Default border color.
    *   **Width**: 0px - 5px.
    *   **Radius**: Rounding of input corners (0px - 30px).
    *   **Focus Border Color**: Highlight color when the user clicks into the field.
*   **Text Styling**:
    *   **Text Color**: Color of the user's typed text.
    *   **Font Size**: 10px - 20px.
    *   **Placeholder Color**: Color of the hint text shown before typing.
*   **Padding**: Inner spacing for comfortable typing (4px - 20px).
*   **Input Size**: Relative width of the input box (Small 50%, Medium 75%, Large 100%).

### 7. Button Settings
Customize the **Submit** and **Reset** buttons independently.

*   **Select Button**: Switch editing focus between 'Submit' and 'Reset'.
*   **Button Label**: Rename the button text (e.g., Change "Submit" to "Send Now").
*   **Background Style**:
    *   **Solid**: Single color.
    *   **Gradient**: Two-color gradient.
    *   **Outlined**: Transparent background with colored border.
*   **Dimensions**:
    *   **Button Size**: Small, Medium, Large (min-width presets).
    *   **Padding**: Custom Horizontal and Vertical padding (0-50px).
*   **Styling**:
    *   **Text Color**: Font color.
    *   **Border Radius**: 0px - 50px (create pill-shaped buttons).
    *   **Border Thickness**: 0px - 10px.
    *   **Border Color**: Custom border color.
    *   **Font Size**: 10px - 24px.
